# Immersive Social Portfolio

A premium personal portfolio inspired by modern social media interfaces, featuring 3D animations and a custom story highlight system.

## Key Features

- **3D Hero Transformation**: Scroll-triggered GSAP flip animation.
- **Story System**: Custom story viewer with 3D transitions and auto-advance.
- **Dynamic Data**: Content managed via Supabase (Postgres & Storage).
- **Premium UI**: Monochromatic design with smooth micro-animations.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Supabase (Database & Storage)
- **Animation**: GSAP, Framer Motion
- **Styling**: Vanilla CSS

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   Create a `.env` file with your Supabase credentials (see `.env.example`).

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
