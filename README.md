# Portfolio Project

A full-stack portfolio website featuring a React frontend and a Python backend.

## Project Structure

This project consists of two main components:

- **backend/**: Python backend service
- **portfolio/**: React TypeScript frontend application

## Backend

The backend is a Python application built with modern Python practices.

### Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   pip install -e .
   ```

3. Run the application:
   ```bash
   python main.py
   ```

## Frontend

The frontend is a React application built with TypeScript and Vite.

### Setup

1. Navigate to the portfolio directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

5. Preview the production build:
   ```bash
   npm run preview
   ```

## Development

- Backend: Python 3.13+
- Frontend: Node.js with npm

## Technologies Used

- **Backend**: Python
- **Frontend**: React, TypeScript, Vite
- **Styling**: CSS (expand as needed)
- **Linting**: ESLint
